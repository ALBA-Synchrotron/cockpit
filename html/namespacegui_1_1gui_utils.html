Add a labeled form input to the given sizer.

Note that if you pass in a wx.CheckBox input, then the input and the explanatory text are swapped.

Parameters
defaultValueThe default value the form input should have.
labelHeightAdjustmentNumber of pixels to push the input's label down by. Sometimes labels and inputs don't align nicely otherwise.
controlTypeThe type of the form input to make. Defaults to wx.TextCtrl.
controlThe specific control object to use. Mutually exclusive with the controlType parameter.
helperStringHelp text to insert, using addHelperString.
flagsAny wx flags to use when inserting the object into the sizer.
