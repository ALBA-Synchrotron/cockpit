We allow either for a set of pre-chosen filters (via DrawerSettings), or for a more variable approach with callbacks.

If callbacks are supplied, they override the DrawerSettings if any.

Parameters
settingsA list of DrawerSettings instances.
settingIndexIndex into settings list indicating the current mode.
callbacksIf available, must implement:
  • getWavelengthForCamera(name, cameraName): return the wavelength of light seen by that camera.
  • getDyeForCamera(name, cameraName): return the name of the dye the camera sees, or some other textual description (e.g. "Full")
  • getColorForCamera(name, cameraName): return the color used to represent the named camera.
