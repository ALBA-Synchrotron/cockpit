def gui.macroStage.macroStageBase.MacroStageBase.refreshWaiter (   self)

Wait until a minimum amount of time has passed since our last redraw before we redraw again.

Since we redraw when the stage moves or the step size changes, and these events may happen rapidly, this prevents us from spamming OpenGL calls.

Rescale the input value to be in the range (self.min[X|Y], self.max[X|Y]), flip the X axis, and call glVertex2f on the result.

Or return the value instead if shouldReturn is true. The basic idea here is to go from stage coordinates to view coordinates.
