  • setPosition(name, index): Set the filter's position.
    Parameters
    filterAmountsAn array of filtration amounts. If this is a global filter (affecting all light sources), then this is a 1D array; otherwise it is an ND array where the first (N-1) indices are the positions of the global filters.
