Detailed Description

This experiment is an example of a custom opto script.

It is identical to a standard Z-stack experiment except that it turns on and off lights in a configurable timed sequence before running the stack Note that this class inherits from the ZStackExperiment class in experiment/zStack.py; the generateActions() function in that class is what collects the actual data. We just modify the execute() function (inherited from the base Experiment class in experiment/experiment.py) to engage the LEDs. See below this class for the ExperimentUI and OptoPanel classes that set up the UI for configuring the experiment settings.
