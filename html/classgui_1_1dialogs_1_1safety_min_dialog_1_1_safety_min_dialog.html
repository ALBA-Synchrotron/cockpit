Detailed Description

This class provides a simple wrapper around the interfaces.stageMover's safety functionality.

Note that unlike most dialogs, this one does not save the user's settings; instead, it always shows the current safety min as the default setting. This is to keep users from blindly setting the safety min to what they always use; we want them to think about what they're doing.
