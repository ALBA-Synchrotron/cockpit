Detailed Description

This class communicates with the digital delay generator, which we use to provide access to some light sources with very fine time resolution.

def devices-omx.delayGen.DigitalDelayGeneratorDevice.examineActions (   self,
  name,
  table 
)

Take a look at the provided table.

We do two main things here. First, we track exposure times so that we're set for the right output during the experiment (and add actions to adjust our exposure times as needed). Second, we replace uses of LightSource handlers with the DSP's delayHandler object, so that we get triggered instead of opening the physical shutters (which are instead left on for the duration of the experiment).
