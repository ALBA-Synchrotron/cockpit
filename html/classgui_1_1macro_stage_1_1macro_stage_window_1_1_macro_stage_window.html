Detailed Description

This class simply contains instances of the various MacroStage subclasses, side-by-side, along with the buttons associated with each.

It also allows for communication between the different subclasses, and has some logic that is generally related to the UIs the MacroStage instances provide but is not tightly bound to any one of them.
