Detailed Description

An important clarification about this system: normally the assumption is that cameras will be driven by external trigger.

Thus there's no "take image" function here. Some external software is expected to notice when the camera has taken an image and send it to us here, at which point it is propagated to the rest of the cockpit. Because this is a dummy camera, we don't have exactly that system in place.
