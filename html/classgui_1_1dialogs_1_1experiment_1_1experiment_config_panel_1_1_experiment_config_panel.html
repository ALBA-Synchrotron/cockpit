Detailed Description

This class provides a GUI for setting up and running experiments, in the form of an embeddable wx.Panel and a selection of functions.

To use the panel, create an instance of ExperimentConfigPanel and insert it into your GUI. Call its runExperiment function when you are ready to start the experiment.

The parent is required to implement onExperimentPanelResize so that changes in the panel size will be handled properly.
