Detailed Description

This class collects the data needed to perform response-map correction.

It takes exposures of some flat illumination source (e.g. a Chroma slide) at varying exposure times, to characterize the response of the cameras to incident light. Structurally this class is very similar to the OffsetGainCorrectionExperiment class.
