callbacks must include the following:

  • examineActions(name, table): Perform any necessary validation or modification of the experiment's ActionTable.
  • getNumRunnableLines(name, table, index): Given an input ActionTable, return the number of lines in the table that we can implement, starting from the specified index.
  • executeTable(name, table, startIndex, stopIndex): Actually perform actions through the specified lines in the ActionTable.
