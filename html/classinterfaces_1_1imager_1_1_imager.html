Detailed Description

This module provides an interface for taking images with the current active cameras and light sources.

It's used only outside of experiment mode. Simple container class.

def interfaces.imager.Imager.videoMode (   self)

Video mode: continuously take images at our maximum update rate.

We stop whenever the user invokes takeImage() manually or the abort button is pressed. We also limit our image rate if there are any non-room-light light sources to 1 image per second, to avoid excessive sample damage.
