Construct any special UI the Device needs.

Most Devices will not need to do anything here, but if you have settings that the user needs to be able to manipulate and that the normal UI will not handle, then this is where you create your specific UI.

Returns
a WX Sizer or Panel that will be inserted into the main controls window, or None if nothing is to be inserted.
